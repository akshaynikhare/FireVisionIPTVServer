GitHub Releases Management

APK versions are now managed via GitHub Releases in the akshaynikhare/FireVisionIPTV repository.

  1. Create a new release in the FireVisionIPTV GitHub repo.
  2. Upload the signed APK as a release asset.
  3. Update APP_VERSION_NAME and APP_VERSION_CODE in the server env.

The server automatically exposes the latest release via /api/v1/app endpoints and uses the direct GitHub download URL.
