JSON-Based Management

APK versions are managed via the versions.json file. Use one of these methods:

Method 1: Helper Script (Recommended) 
node add-version.js path/to/app.apk 1.1.0 2 "Release notes"
Method 2: Manual
  1. Copy APK to apks/ directory
  2. Edit versions.json file
  3. Restart server

See QUICK_START_APK.md for details
